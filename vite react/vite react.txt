💡 Vite kya hai?

Vite ek build tool aur development server hai, jo modern frontend frameworks jaise React, Vue, etc. ke sath use hota hai.


🔧 Vite ka React ke sath kya kaam hai?

React app banate waqt, humein ek development environment chahiye hota hai jisme:

Code fast compile ho

Browser me turant changes dikhen (Hot Reloading)

Final build optimized ho


🔥 Vite vs CRA (Create React App)

Pehle React projects ke liye log Create React App (CRA) use karte the. Lekin CRA thoda slow ho jaata tha bade projects me.



                                                🛠️ Step-by-Step Installation

🎯 VITE + REACT: INSTALLATION GUIDE (FULL STEPS)
===============================================

🟢 1. CHECK PREREQUISITES
--------------------------
- Make sure Node.js is installed (v16+ recommended)
- To check versions, run:
    node -v
    npm -v

📦 Install Node.js if needed: https://nodejs.org/


🟦 2. CREATE A NEW PROJECT
---------------------------
Run one of the following commands in your terminal:

Using NPM:
    npm create vite@latest my-react-app -- --template react

Using Yarn:
    yarn create vite my-react-app --template react

Using PNPM:
    pnpm create vite my-react-app -- --template react

* Replace "my-react-app" with your desired folder name.


🟨 3. NAVIGATE TO PROJECT FOLDER
---------------------------------
    cd my-react-app


🟠 4. INSTALL DEPENDENCIES
---------------------------
Using NPM:
    npm install

Using Yarn:
    yarn

Using PNPM:
    pnpm install


🟣 5. START DEVELOPMENT SERVER
-------------------------------
    npm run dev
    # or
    yarn dev
    # or
    pnpm dev

🌐 Open your browser and visit:
    http://localhost:5173


🔵 6. BUILD FOR PRODUCTION (Optional)
--------------------------------------
    npm run build
    # or
    yarn build
    # or
    pnpm build

This creates a "dist/" folder with production-ready files.


📁 7. PROJECT STRUCTURE OVERVIEW
---------------------------------
my-react-app/
├─ public/
├─ src/
│  ├─ App.jsx
│  ├─ main.jsx
├─ index.html
├─ package.json
├─ vite.config.js


✅ YOU ARE NOW READY TO DEVELOP WITH VITE + REACT!


💡 EXTRA TIPS (Optional Installs)
----------------------------------
- React Router DOM:
    npm install react-router-dom

- Tailwind CSS:
    Follow Tailwind setup guide: https://tailwindcss.com/docs/guides/vite

- TypeScript Template:
    npm create vite@latest my-app -- --template react-ts
